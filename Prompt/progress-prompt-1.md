🟣 PROMPT 1 — Setup inițial & structura proiectului
Vreau să construim de la zero o aplicație web pentru HR (Resurse Umane)
pentru lucrarea mea de licență.

TITLU LUCRARE: "Soluție web/mobilă pentru eficientizarea activităților
din departamentul de Resurse Umane"

STACK ALES:
- Frontend: React.js + Tailwind CSS + React Router
- Backend: Node.js + Express.js
- Baza de date: PostgreSQL
- ORM: Prisma
- Autentificare: JWT

FUNCȚIONALITĂȚI:
1. Autentificare cu 3 roluri: Admin HR, Manager, Angajat
2. Gestionare angajați (CRUD complet)
3. Cereri de concediu (depunere + aprobare/respingere)
4. Pontaj / evidența orelor lucrate
5. Documente HR (upload + vizualizare)
6. Notificări interne
7. Rapoarte și statistici cu grafice
8. Modul recrutare (joburi + candidați)

CERINȚĂ PENTRU COD:
- Generează structura completă a proiectului (toate folderele și fișierele)
- Explică fiecare folder și fișier ce face
- Cod curat, comentat în română
- Generează toate comenzile de terminal necesare

CERINȚĂ SPECIALĂ — PAȘI DE VERIFICARE:
După fiecare bloc de cod sau comandă, adaugă o secțiune
"✅ CE TREBUIE SĂ FAC EU" care să conțină:
- Exact ce comenzi scriu în terminal
- Ce fișiere deschid în VS Code
- Ce ar trebui să văd pe ecran dacă totul merge bine
- Ce înseamnă dacă apare o eroare comună și cum o rezolv

CERINȚĂ SPECIALĂ — DOCUMENTAȚIE PROGRES:
La final, generează un bloc markdown intitulat:
"📄 DOCUMENT PROGRES — PROMPT 1 FINALIZAT"
care să conțină:
- Ce s-a construit în acest prompt (listă scurtă)
- Structura folderelor creată
- Versiunile exacte ale pachetelor instalate
- Variabilele de mediu configurate
- Porturile folosite (frontend/backend/DB)
- Status: ✅ Funcțional / ❌ Necesită verificare
Acest document îl voi adăuga ca și context la promptul următor.
